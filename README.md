🤖 AI-Powered CV Analysis

A smart and efficient project built using Python that leverages Artificial Intelligence (AI) and Natural Language Processing (NLP) to analyze resumes (CVs).
This project helps recruiters and students by automatically extracting key details from CVs and matching them with job requirements.

🌐 Live Demo---> https://streamweb45.streamlit.app


📌 Features
-----------------------------------------------------------------------------------------------------

📄 Automatic CV Parsing – Extracts skills, education, and work experience.

🧠 AI-Powered Analysis – Uses NLP and ML algorithms to evaluate resumes.

🎯 Candidate Scoring – Matches CVs against job descriptions.

⚡ Fast & Efficient – Saves recruiters time by automating manual screening.

📊 Insightful Reports – Highlights strengths and gaps in candidate profiles.





🛠️ Technologies Used
-----------------------------------------------------------------

Python

NLTK / spaCy (for text processing)

scikit-learn (for ML models)

Pandas & NumPy (for data handling)

Matplotlib / Seaborn (for visualization, optional)
